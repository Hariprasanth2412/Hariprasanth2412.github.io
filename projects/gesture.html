AI Gesture & Voice Control System

Project Overview

An AI-powered desktop automation system that allows users to control their computer using hand gestures and voice commands. The project uses MediaPipe, OpenCV, Speech Recognition and PyAutoGUI to provide a touch-free user experience.

Features

  • ✔ Hand Gesture Recognition
  • ✔ Voice Command Support
  • ✔ Mouse Control
  • ✔ Open Applications
  • ✔ Close Windows
  • ✔ Desktop Automation

Technologies Used

Python OpenCV MediaPipe PyAutoGUI SpeechRecognition

Challenges

Implemented smooth mouse movement, optimized gesture detection, reduced lag and improved accuracy using MediaPipe hand landmarks.

What I Learned

Working with Computer Vision, Artificial Intelligence, Human Computer Interaction and Python automation.
